# 💻 Dell Inventory Management System

A Dell-themed Inventory Management System built using **HTML, CSS, and JavaScript**. The application allows users to manage Dell products efficiently with CRUD operations and real-time stock availability checking.

## 🚀 Features

- Display current inventory of Dell products
- Add new products to inventory
- Remove existing products
- Check product availability
- CRUD operations using JavaScript arrays
- Responsive Dell-inspired user interface
- Event handling with JavaScript functions

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
